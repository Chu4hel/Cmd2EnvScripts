# Утилиты командной строки

Этот набор скриптов предназначен для упрощения работы в командной строке Windows.

## Быстрый старт

Чтобы использовать эти скрипты из любой папки в `cmd`, необходимо добавить директорию со скриптами в системную переменную `PATH`.

1.  Откройте командную строку (`cmd`).
2.  Выполните следующую команду, чтобы добавить путь к папке со скриптами в `PATH` для вашего пользователя:

    ```shell
    setx PATH "%PATH%;D:\Cmd2EnvScripts"
    ```

3.  **Важно:** Перезапустите командную строку, чтобы изменения вступили в силу.

## Описание скриптов

| Команда      | Описание                                                                                              |
| :----------- | :---------------------------------------------------------------------------------------------------- |
| `b.cmd`      | Переходит в родительскую директорию (`cd ..`).                                                        |
| `env.cmd`    | Открывает окно "Переменные среды" для редактирования.                                                 |
| `find-cmd.cmd` | Ищет команды, доступные в системе, по части имени. Например: `find-cmd code`.                         |
| `pathc.cmd`  | Копирует текущий путь (`pwd`) в буфер обмена.                                                         |
| `services.cmd` | Открывает оснастку "Службы" (`services.msc`).                                                         |
